Для чего используются типы данных: json, xml, yaml?

Json:
Применяется при передаче данных с клиент<->сервер
Для сериализации объектов
Для создания файлов конфигурации
Везде, где применяется JavaScript, Python. Подходит для сериализации составных структур данных из списков и словарей

yaml:
Для сериализации объектов
Для создания файлов конфигурации
Везде, где применяется JavaScript, Python. Подходит для сериализации составных структур данных из списков и словарей

xml:
Применяется при передаче данных с клиент<->сервер
Для сериализации объектов
Для создания файлов конфигурации
Применяется повсюду
Подходит для сериализации сложных бизнес-объектов
